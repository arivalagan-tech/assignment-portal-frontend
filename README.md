# 📋 Assignment Portal Frontend

A modern, high-performance **React.js** frontend application for managing assignment workflows with real-time synchronization, advanced state management, and responsive UI design.

---

## ✨ Features

- ⚡ Lightning-Fast Performance – Optimized React with Context API
- 🎨 Beautiful UI – TailwindCSS with dark mode support
- 📱 Mobile-First Design – Fully responsive
- 🔄 Real-Time Updates – Live synchronization
- ♿ Accessibility – WCAG 2.1 compliant
- 🔐 Secure Authentication – Best practices implemented

---

## 🛠 Tech Stack

- **React 18+** – Modern React with hooks
- **TailwindCSS 3+** – Utility-first CSS framework
- **Context API** – State management
- **React Router v6** – Client-side routing
- **Vite/CRA** – Fast build tooling


---

## 🚀 Quick Start

```bash
# Clone & Install
git clone https://github.com/arivalagan-tech/assignment-portal-frontend.git
cd assignment-portal-frontend
npm install

# Development
npm run dev

# Build
npm run build

```

---

## 📁 Project Structure

```
src/
├── components/     # Reusable components
├── pages/          # Route pages
├── context/        # Context API setup
├── hooks/          # Custom hooks
├── services/       # API calls
├── utils/          # Utilities

```
